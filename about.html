What is HTML?

HTML stands for Hyper Text Markup Language and is primarily used for conducting and designing websites.

Basic Features:

  • !DOCTYPE HTML
  • HTML
  • Head
  • Title
  • Meta
  • Link
  • Style
  • Script
  • H1, H2, H3, etc. (heading)
  • P (paragraph)
  • Body

What is CSS?

CSS statnds for Cascading Style Sheets and is primarily used to design the look and layout of the website.

Basic Features:

  • Background Color
  • Paddding
  • Font-Family
  • Font-Weight
  • Font-Size
  • Display
  • Margin
